오늘 한 것

git : 코드 변경점 기록 (버전관리도구 / 형상관리도구 / 소프트웨어의 변경사항을 체계적으로 추적하고 통제)
git hub : 온라인 코드 저장소 (온라인 백업, 공유, 협업)

  • 팀 소통공간 개설
  • 와이어 프레임 마무리 / 서로 공유하기
  • git 특강
    1. pwd : 현재 작업환경 확인
    2. 파일들을 확인하는 ls와 숨겨진 파일 설정들도 확인할 수 있는 ls-a [li는 리스트의 약자, -a의 a는 all을 의미한다]
    3. cd 폴더명 : 폴더 이동.
    4. mkdir 폴더명: 폴더 만들기
    5. touch 파일명: 파일 만들기
