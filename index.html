강의 내용 정리
git : 코드 변경 기록 (버전관리도구 / 형상솬리도구 / 소프트웨어의 변경사항을 체계적으로 추적하고 통제)
git hub : 온라인 코드 저장소 (온라인 백업, 공유, 협업)
    😈 - git 특강
  • pwd : 현재 작업환경 확인
  • ls-a : 숨겨진 파일 설정들도 확인
  • cd 폴더명 : 폴더 이동
  • mkdir 폴더명 : 폴더 생성
  • touch 파일명 : 파일 생성
개발 단계 정리
1. 팀 이름 만들기 - 코딩쉽지않조
2. 기본 셋팅 맞추기 (prettier, esLint 등) 3. 피그마로 와이어 프레임 잡아오지
